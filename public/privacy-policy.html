02 Bagaimana Data Digunakan

Data yang dikumpulkan digunakan semata-mata untuk:

  • Menjalankan sistem ekonomi virtual (saldo, transaksi, cooldown)
  • Menyimpan progress dan riwayat aktivitas pengguna
  • Menampilkan statistik dan leaderboard komunitas
  • Mengirim broadcast update ke server yang terdaftar
  • Mendeteksi dan mencegah penyalahgunaan layanan

Data kamu tidak pernah digunakan untuk keperluan iklan, profiling komersial, atau tujuan di luar fungsi bot.

03 Penyimpanan Data

Semua data disimpan menggunakan Cloudflare KV — layanan penyimpanan terenkripsi milik Cloudflare, Inc. Data disimpan di infrastruktur Cloudflare yang tersebar secara global dengan keamanan tingkat enterprise.

  • Data disimpan selama pengguna masih aktif menggunakan layanan
  • Data server dihapus otomatis jika bot dikeluarkan dari server
  • Pengguna dapat meminta penghapusan data kapan saja

04 Berbagi Data dengan Pihak Ketiga

OWO BIM tidak menjual, menyewakan, atau membagikan data pengguna kepada pihak ketiga untuk tujuan komersial.

Data hanya dapat diberikan kepada pihak lain dalam kondisi berikut:

  • Diwajibkan oleh hukum atau perintah pengadilan yang berlaku
  • Diperlukan untuk melindungi keamanan pengguna lain atau publik
  • Infrastruktur pihak ketiga yang digunakan (Cloudflare) sebagai penyedia layanan teknis

Kebijakan privasi Cloudflare dapat dilihat di cloudflare.com/privacypolicy.

05 Keamanan Data

Kami menerapkan langkah-langkah keamanan untuk melindungi data kamu:

  • Password disimpan dalam bentuk hash SHA-256 yang tidak dapat dibalik
  • Semua komunikasi menggunakan enkripsi HTTPS/TLS
  • Akses ke data admin dibatasi dengan verifikasi Discord ID
  • Tidak ada data sensitif yang disimpan dalam log atau cache

Meskipun kami menerapkan keamanan terbaik, tidak ada sistem yang 100% aman. Jika kamu menemukan celah keamanan, harap laporkan segera kepada kami.

06 Hak Pengguna

Sebagai pengguna, kamu memiliki hak berikut:

  • Akses — Mengetahui data apa yang kami simpan tentang kamu

  • Penghapusan — Meminta penghapusan seluruh data kamu dari sistem
  • Portabilitas — Meminta salinan data kamu dalam format yang dapat dibaca

Untuk menggunakan hak-hak ini, hubungi kami melalui kontak yang tersedia di bagian bawah halaman ini.

07 Data Anak-anak

Kami tidak secara sengaja mengumpulkan data dari anak-anak di bawah usia minimum. Jika kamu percaya bahwa data anak di bawah umur telah dikumpulkan, harap hubungi kami segera untuk penghapusan.
